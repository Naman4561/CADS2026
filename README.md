# HABI_v2_exoplanets


## Overview


The Objective of this project is to discover habitable exoplanets and assess their potential for supporting life. This was completed for the Zero Gravity data challenge hackathon that was hosted by University of North Carolina's computer science department.

## Requirements and setup

This repository includes a pinned dependency file: `requirements.txt`. To reproduce the analysis, create an isolated Python environment and install the dependencies from that file.

Recommended approaches:

- Using Python venv (Windows PowerShell):

```
python -m venv env
.\env\Scripts\Activate.ps1
pip install -r requirements.txt
```

- Using pip in an existing (virtual) environment:

```
pip install -r requirements.txt
```

- Using conda:

```
conda create -n habi python=3.10; conda activate habi; pip install -r requirements.txt
```

If you prefer to install packages manually, the main packages used in the analysis include: numpy, pandas, matplotlib, seaborn, scikit-learn, astropy, astroquery, and jupyterlab. The `requirements.txt` file pins tested versions; update it with `pip freeze > requirements.txt` if you make changes to the environment.

Note: The environment was developed against Python 3.10+; compatibility with later Python versions is likely but not guaranteed without testing.

## Running the notebook

1. Start Jupyter Lab or Notebook:

```
jupyter lab
# or
jupyter notebook
```

2. Open `HABI_v2_exoplanets.ipynb` and execute cells interactively.

Optional: export results to HTML:

```
jupyter nbconvert --to html HABI_v2_exoplanets.ipynb
```
